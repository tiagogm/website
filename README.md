This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/zeit/next.js/tree/canary/packages/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `pages/index.js`. The page auto-updates as you edit the file.

## Theme System

This project includes a dark mode theme system with automatic OS preference detection and manual toggle. See [docs/theme.md](./docs/theme.md) for detailed documentation on how the theme system works and how to extend it.
